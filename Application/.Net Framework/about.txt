Writing C# Programs on Linux or Mac OS
Although the.NET Framework runs on the Windows operating system, there are some alternative versions that work on other operating systems. 
Mono is an open-source version of the .NET Framework which 
	includes a C# compiler and runs on several operating systems, 
	including various flavors of Linux and Mac OS.

Purpose of Mono
	able to run Microsoft .NET applications cross-platform, 
	provide better development tools for Linux developers. 


Mono can be run on many operating systems including Android, BSD, iOS, Linux, OS X, Windows, Solaris, and UNIX.
